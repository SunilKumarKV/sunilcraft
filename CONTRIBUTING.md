# Contributing to SunilCraft

Thank you for contributing.

## Local Setup

```bash
npm install
npm run dev
```

## Contribution Rules

- Do not commit real environment variables or credentials.
- Keep portfolio branding professional.
- Ensure project/demo links remain valid.
- Test responsive UI changes.
- Keep recruiter-facing content clean and accurate.

## Pull Requests

Use the pull request template and include screenshots for UI changes.
